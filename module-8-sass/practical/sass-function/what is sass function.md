# what is sass function  ?
 1. A sass function basically used as a block of code 
 2. sass function is used to return a values 
 3. sass function is a block of code that can be used to completed any task that i.e called sass function 
## advantage to used sass function 
 1. sass function are re-usables 
 2. sass function is used calculate and return a single value(number, color, string , list , map etc)
 3. sass function return mixin as a function in css 
 4. a sass function do not return a css directly we can return in css by using @return keyword 

 **syntax of sass function**

  ```
  @function function-name(arg1,arg2,arg3)
  {
    @return value
  } 

  ```
